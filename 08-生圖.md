# OpenCode 懶人包 #08：生圖技能

> 版本：v0.1
> 更新日期：2026-05-19

---

## 這個懶人包會幫你做什麼？

讓 OpenCode 可以畫圖 — 幫你安裝 draw skill：

- 設定 OpenAI API Key
- 安裝 draw skill（gpt-image-2 生圖）
- 畫出第一張圖驗證
- 完成後在任何專案說「畫一張 XX」就能生圖

---

## 先備條件

- [ ] 已完成 **懶人包 #00：環境建置**
- [ ] 已有 OpenAI API Key（或使用 opencode 內建生圖能力）

---

## 請 OpenCode 幫我執行以下步驟

### 步驟一：確認可用生圖方式

OpenCode 支援多種生圖方式。檢查你的 `opencode.json` 有無 provider 設定。最簡單的方式是直接使用 OpenCode 內建的 provider。

如果需要自訂生圖技能：

### 步驟二：安裝 draw skill

建立 `~/.config/opencode/skills/draw/SKILL.md`：

---
name: draw
description: OpenAI gpt-image-2 生圖 — 說「畫一張」「生圖」時載入
---

當使用者要求生圖時：
1. 確認圖片規格（主題、風格、尺寸）
2. 使用 OpenAI gpt-image-2 模型
3. 根據複雜度自動決定 quality（low/medium/high）
4. 存檔至當前專案 `slides/generated/` 目錄
5. 回報使用等級與費用

---

## 驗證

重新開啟 OpenCode，然後說：

```
畫一張貓咪坐在鋼琴上的插圖，粉筆黑板風格。
```
