# OpenCode 懶人包 #08：生圖技能

> 版本：v0.2
> 更新日期：2026-05-25

---

## 這個懶人包會幫你做什麼？

讓 OpenCode 可以畫圖 — 幫你安裝 draw skill（OpenAI gpt-image-2 生圖）：

- 設定 OpenAI API Key
- 安裝 draw skill 檔案（SKILL.md + draw.py）
- 設定 opencode.json 權限
- 測試生圖
- 完成後在任何專案說「畫一張 XX」就能生圖

---

## 先備條件

- [ ] 已完成 **懶人包 #00：環境建置**
- [ ] Python 3.10+ 已安裝
- [ ] 已有 OpenAI API Key
- [ ] `openai` Python 套件已安裝：`pip install openai`

---

## 請 OpenCode 幫我執行以下步驟

### 步驟一：設定 API Key

建立 `~/.openai.env`：

```
OPENAI_API_KEY=sk-proj-你的金鑰
```

確認：
```bash
type %USERPROFILE%\.openai.env
```

### 步驟二：安裝 draw skill 檔案

建立目錄並複製技能檔案：

```bash
mkdir -p ~/.config/opencode/skills/draw
```

從本 repo 複製 SKILL.md：
```bash
curl -o ~/.config/opencode/skills/draw/SKILL.md https://raw.githubusercontent.com/mathruffian-dot/opencode-lazy-packs/main/skills/08-draw/SKILL.md
```

複製 draw.py（若已有則跳過）：
```bash
curl -o ~/.config/opencode/skills/draw/draw.py https://raw.githubusercontent.com/mathruffian-dot/opencode-lazy-packs/main/scripts/draw.py
```

### 步驟三：設定 opencode.json 權限

編輯 `~/.config/opencode/opencode.json`，在 `permission.skill` 加入：

```json
"draw": "allow"
```

### 步驟四：測試生圖

```bash
python ~/.config/opencode/skills/draw/draw.py "一隻橘貓坐在窗邊看夕陽，水彩風格" --name test-cat --quality low
```

檢查輸出：
```bash
ls slides/generated/test-cat_*.png
```

### 步驟五：驗證

重啟 OpenCode 後說：
```
畫一隻貓咪坐在鋼琴上的插圖，粉筆黑板風格。
```

| 等級 | 費用（約 NT$/張） | 適用情境 |

| low | ~$0.3 | 99% 情境（簡報、教學、封面、過場）|

## 常見問題

| 問題 | 解法 |
|------|------|
| 畫圖失敗 | 確認 `~/.openai.env` 的 API key 有效 |
| `openai` module not found | `pip install openai` |
| 403 Organization must be verified | 去 platform.openai.com/settings 做 Individual 驗證 |
| 429 Rate limit | OpenAI 額度用完，去 Billing 儲值 |
