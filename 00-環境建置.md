# OpenCode 懶人包 #00：環境建置

## 這個懶人包會幫你做什麼？

把使用 OpenCode 懶人包所需的環境一次建置完成：
- 安裝 Node.js（如果未安裝）
- 安裝 OpenCode CLI
- 安裝 Git、GitHub CLI、uv
- 驗證所有工具正常運作

---

## 先備條件

- [ ] 電腦有網路連線

---

## 請 OpenCode 幫我執行以下步驟

> 把這份檔案的內容貼給 OpenCode，它會自動執行。

---

### 步驟一：環境檢查

請 OpenCode 先確認以下所有項目：

1. 作業系統：Windows / macOS / Linux
2. 網路連線
3. **Node.js**：`node --version`（需 18+）
4. **Git**：`git --version`
5. **GitHub CLI**：`gh --version`
6. **uv**（Python 工具管理器）：`uv --version`

把缺少的記錄下來，後續步驟逐一安裝。

---

### 步驟二：安裝 Node.js（如果未安裝或版本 < 18）

**Windows**：
```bash
winget install --id OpenJS.NodeJS --accept-source-agreements --accept-package-agreements
```

**macOS**：
```bash
brew install node
```

**Linux（Ubuntu/Debian）**：
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
```

確認：`node --version`

---

### 步驟三：安裝 OpenCode

```bash
npm install -g opencode-ai
```

確認版本：
```bash
opencode --version
```

---

### 步驟四：安裝 Git（如果未安裝）

**Windows**：
```bash
winget install --id Git.Git --accept-source-agreements --accept-package-agreements
```

**macOS**：`xcode-select --install`

**Linux**：`sudo apt update && sudo apt install git -y`

確認：`git --version`

---

### 步驟五：安裝 GitHub CLI（如果未安裝）

**Windows**：
```bash
winget install --id GitHub.cli --accept-source-agreements --accept-package-agreements
```

**macOS**：`brew install gh`

**Linux（Ubuntu/Debian）**：
```bash
sudo apt update && sudo apt install gh -y
```

**Linux（Fedora）**：
```bash
sudo dnf install gh
```

**Linux（Arch）**：
```bash
sudo pacman -S github-cli
```

確認：`gh --version`

---

### 步驟六：安裝 uv（如果未安裝）

**Windows（PowerShell）**：
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

**macOS / Linux**：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

確認：`uv --version`

---

### 步驟七：最終驗證

```bash
node --version
opencode --version
git --version
gh --version
uv --version
```

> ✅「環境建置完成！你的電腦已經準備好使用所有 OpenCode 懶人包了。」

## 常見問題

| 問題 | 解法 |
|------|------|
| `opencode` 指令找不到 | 確認 npm global bin 在 PATH 中：`npm config get prefix`，把 `prefix/bin` 加入 PATH |
| winget 找不到 | Windows 10 1809+ 內建；或到 Microsoft Store 安裝「App Installer」 |
| Node.js 版本太舊 | 移除舊版重裝最新 LTS |
