# OpenCode 懶人包 #06：安裝瀏覽器控制

> 版本：v0.1
> 更新日期：2026-05-19

---

## 這個懶人包會幫你做什麼？

讓 OpenCode 可以控制瀏覽器和桌面應用：
- 安裝 Playwright MCP（網頁控制）
- 安裝 open-computer-use（桌面自動化）
- 寫入 opencode.json 設定
- 驗證連線

---

## 先備條件

- [ ] Node.js 18+ 已安裝
- [ ] 電腦有網路

---

## 請 OpenCode 幫我執行以下步驟

### 步驟一：寫入 Playwright MCP 設定

編輯 `~/.config/opencode/opencode.json`，加入：

```json
{
  "mcp": {
    "playwright": {
      "type": "local",
      "command": ["npx", "-y", "@playwright/mcp"],
      "enabled": true
    }
  }
}
```

### 步驟二：寫入 open-computer-use 設定

```json
{
  "mcp": {
    "open-computer-use": {
      "type": "local",
      "command": ["npx", "-y", "open-computer-use"],
      "enabled": true
    }
  }
}
```

---

### 步驟三：驗證 Playwright

重新開啟 OpenCode，然後問它：

```
請開啟 https://example.com 並告訴我頁面標題是什麼。
```

### 步驟四：驗證 open-computer-use

```
請截圖我的桌面（或特定視窗）。
```

---

## 完成回報格式

```md
## 瀏覽器控制安裝完成

- Playwright MCP：已設定 / 已啟用
- open-computer-use：已設定 / 已啟用
- 瀏覽器導航測試：成功 / 失敗
- 桌面截圖測試：成功 / 失敗
```

---

## 常見問題

| 問題 | 解法 |
|------|------|
| Playwright 首次執行下載瀏覽器很慢 | 等待下載完成，只需一次 |
| open-computer-use 找不到視窗 | 確認目標應用正在執行 |
